# FMPXMLRESULT.php
Convert PHP array to FMPXMLRESULT

Example use
```
$tmp = array(
	'fieldname1' => 'fieldname1Content',
	'fieldname2' => 'fieldname2Content'
);

MakeFMPXMLRESULTFromArray( $tmp );
```
